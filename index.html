iPogo Installer

Instalador para Linux — sideload sem complicação

Linux x86_64 · Ubuntu / Debian Baixar Instalador

Requisitos

  • Linux Ubuntu / Debian (x86_64)
  • Docker instalado
  • iPhone conectado via cabo USB
  • Apple ID (com ou sem 2FA)

Como usar

1

Baixe o instalar.sh acima, abra o terminal na pasta e rode:
bash instalar.sh
Isso instala o app no menu de aplicativos automaticamente.

2

Conecte o iPhone, baixe o .ipa em ipogo.app e selecione na interface

3

Digite seu Apple ID e senha. Se tiver 2FA, o código do iPhone aparece automaticamente

4

No iPhone: Ajustes → Geral → VPN e Gerenciamento → Confiar
